# لعبة توصيل الأدوية 🚚💊

لعبة بسيطة وممتعة لتوصيل الأدوية من المخزن للصيدليات.

## 🎮 طريقة اللعب
1. اضغط على **ابدأ اللعبة**
2. اختار الدواء من القائمة
3. اضغط على الشاحنة 🚚 عشان توصل الدواء
4. كل ما توصل أسرع، تكسب نقاط ومكافآت أكثر

## 🏆 المكافآت
- 🌟 نجمة برونزية: 5 توصيلات
- ⭐ نجمة فضية: 10 توصيلات  
- ✨ نجمة ذهبية: 15 توصيلة
- 👑 تاج التميز: 20 توصيلة

## 🛠️ التقنيات المستخدمة
- HTML
- CSS
- JavaScript
